# Security policy

## Supported versions

Security fixes are provided for the latest released version of this
integration. Please update to the newest release before reporting an issue.

## Reporting a vulnerability

Please report security vulnerabilities through GitHub Security Advisories for
this repository. Do not open public issues or discussions.

Include as much detail as you can:

- A clear description of the issue and potential impact
- Steps to reproduce
- Affected versions
- Any relevant logs or traces (redact credentials and personal data)

## Disclosure process

Reports are acknowledged as soon as possible and a fix is prepared. When a fix
is available, the disclosure timeline is coordinated with you.
